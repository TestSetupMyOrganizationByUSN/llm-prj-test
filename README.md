# AI 맛집 추천 웹 애플리케이션

## 개요

다양한 LLM API를 활용하여 사용자 맞춤 맛집 추천 및 이미지 생성 기능을 제공하는 웹 애플리케이션입니다.

## 기능

- 사용자 입력 텍스트 기반 맛집 추천
- 맛집 추천에 어울리는 이미지 생성
- 맛집에 대한 상세 설명 제공

## 기술 스택

- HTML
- CSS
- JavaScript
- Bootstrap
- Fetch API

## 사용 방법

1. 웹 페이지에 접속하여 맛집 추천을 원하는 텍스트를 입력합니다.
2. "추천받기" 버튼을 클릭하면 LLM API를 통해 맛집 추천, 이미지 생성, 설명 생성이 이루어집니다.
3. 맛집 추천 결과와 함께 생성된 이미지 및 설명을 확인합니다.
